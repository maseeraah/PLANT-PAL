BloomBuddy is a simple Python console application that helps you in keeping track of your plants and reminds you when it's time to water them.
Designed as a syllabus-aligned project, it uses skills like problem solving, algorithm design, and python fundamentals while solving a real-world need.

FEATURES
- Add plants with custom watering frequency (in days).
- Track last watered date for each plant.
- Check reminders to see which plants need watering today.
- List all plants with their details.
- User-friendly menu-driven interface.

HOW IT WORKS 
1. Add Plant - Enter plant name and watering frequency.
2. Water Plant - Choose a plant from the list to mark it as watered.
3. Check Reminders - See which plants need watering today.
4. List Plants - View all plants currently stored.
5. Exit - Close the program.

TECH STACK
- Language used: Python 3
- Modules Used: `datetime` (for tracking dates)
- Concepts Applied:
  - Problem solving & top-down design
  - Functions and modularization
  - Lists & dictionaries
  - Control flow (loops, conditionals)
  - Algorithm efficiency
